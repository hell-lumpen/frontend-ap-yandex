# Лекция: Введение в HTML и CSS

## Введение в HTML

### Что такое HTML?

**HTML (HyperText Markup Language)** — это основа любой веб-страницы. Это язык разметки, с помощью которого описывается структура контента. Представьте, что HTML — это каркас дома, который определяет, где будут стены, двери, окна и комнаты. Без HTML не было бы самого здания — веб-страницы.

HTML работает на основе **тегов** — специальных меток, заключённых в угловые скобки, которые браузер понимает как команды, определяющие, что и где должно быть отображено. HTML буквально говорит браузеру: "Эй, этот текст должен быть заголовком, а вот это изображение должно быть расположено тут".

### Зачем нужно знать HTML?

Представьте, что вы архитектор. Чтобы создать функциональное и красивое здание, вам нужно знать, как расположить стены и как они будут работать вместе с окнами и дверьми. То же самое с HTML — зная его, вы можете разрабатывать веб-страницы, на которых элементы расположены логично и эффективно. Неважно, программист вы или дизайнер — понимание основ HTML необходимо каждому, кто хочет иметь дело с веб-технологиями.

### Пример из жизни: Создание визитки

Допустим, вы хотите создать веб-страницу для своего портфолио или бизнес-визитку. С помощью HTML вы можете создать страницу, где будут указаны ваше имя, краткое описание, контактные данные и даже фотографии. HTML позволяет структурировать это так, чтобы каждый элемент был на своём месте и выглядел правильно.

### Основные элементы HTML

#### Теги

1. **Заголовки**: 
   Заголовки помогают пользователю понять, как структурирована страница. HTML поддерживает шесть уровней заголовков — от `<h1>` до `<h6>`. Эти уровни аналогичны разным разделам и подразделам в книге. Например, `<h1>` — это название главы, а `<h2>` — подраздел внутри этой главы. Заголовки также важны для **SEO (поисковой оптимизации)**, поскольку поисковые системы обращают на них внимание при анализе страницы.

   Пример:

   ```html
   <h1>Мой сайт</h1>
   <h2>Обо мне</h2>
   ```

2. **Абзацы**: 
   Текстовые блоки на странице отображаются с помощью тега `<p>`. Это как обычные абзацы в книге. Абзацы помогают читателю легче воспринимать информацию, разделяя её на смысловые блоки.

   Пример:

   ```html
   <p>Я веб-разработчик с опытом работы более 5 лет.</p>
   ```

3. **Ссылки**:
   Ссылки позволяют пользователю перейти на другую страницу или сайт. В реальной жизни ссылки можно сравнить с указателями на дороге, которые ведут вас из одного места в другое.

   Пример:

   ```html
   <a href="https://example.com">Мой блог</a>
   ```

4. **Изображения**:
   Изображения добавляют визуальный контент на веб-страницу, делая её интереснее и живее. Например, на сайте портфолио вы можете разместить фотографии ваших работ.

   Пример:

   ```html
   <img src="photo.jpg" alt="Моё фото">
   ```

#### Структура HTML-документа

Любой HTML-документ делится на две основные части: **`<head>`** и **`<body>`**.

1. **`<head>`** — это техническая часть страницы, которая содержит метаданные, заголовок страницы (то, что отображается на вкладке браузера) и ссылки на подключаемые файлы (например, стили CSS).
   
   Пример:

   ```html
   <head>
       <meta charset="UTF-8">
       <title>Мой сайт</title>
   </head>
   ```

2. **`<body>`** — это всё, что видит пользователь. Здесь размещается контент: заголовки, тексты, изображения, ссылки и другие элементы.

   Пример:

   ```html
   <body>
       <h1>Привет, мир!</h1>
       <p>Это моя первая веб-страница.</p>
   </body>
   ```

Пример полной структуры HTML-документа:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Мой первый сайт</title>
</head>
<body>
    <h1>Добро пожаловать на мой сайт</h1>
    <p>Это первый абзац на моей странице.</p>
</body>
</html>
```

### Научпоп оффтоп: Откуда взялся HTML?

HTML был разработан Тимом Бернерсом-Ли в 1991 году, чтобы учёные могли легко делиться информацией через интернет. С тех пор он значительно эволюционировал, но его суть осталась прежней: создание структуры веб-страниц, которая понятна браузерам и пользователям.

## Введение в CSS

### Что такое CSS?

**CSS (Cascading Style Sheets)** — это язык, который помогает "оживить" HTML. Если HTML — это каркас дома, то CSS — это его интерьер, оформление, выбор цветов, текстур и шрифтов. Он позволяет нам изменять внешний вид элементов, добавлять стили и задавать оформление.

### Зачем нужен CSS?

HTML отвечает за структуру, но без CSS страница выглядела бы скучно и примитивно. CSS позволяет вам сделать веб-страницу красивой и удобной для пользователя. CSS управляет всем: от цвета текста до расстояния между элементами. С его помощью можно задать оформление, которое улучшит восприятие и юзабилити сайта.

### Пример из жизни: Выбор одежды

Представьте, что HTML — это ваше тело, а CSS — это ваша одежда. С помощью CSS вы можете решить, как "одеть" свой сайт: какой цвет выбрать для заголовков, какие шрифты использовать и как разместить изображения. Правильная комбинация CSS-стилей помогает создать уникальный и функциональный дизайн, как и хорошо подобранная одежда создаёт индивидуальный стиль.

### Основные способы использования CSS

1. **Инлайновые стили**:
   Инлайновый стиль применяется непосредственно к HTML-элементу через атрибут `style`. Этот способ подходит для быстрого тестирования или изменения одного конкретного элемента, но не рекомендуется для масштабных проектов, так как сложно управлять стилями большого количества элементов.

   Пример:

   ```html
   <h1 style="color: blue;">Синий заголовок</h1>
   ```

2. **Встроенные стили**:
   Встроенные стили задаются внутри тега `<style>` в секции `<head>`. Этот способ удобен для небольших проектов, но тоже не очень масштабируем.

   Пример:

   ```html
   <head>
       <style>
           h1 {
               color: red;
           }
           p {
               font-size: 18px;
           }
       </style>
   </head>
   ```

3. **Внешние стили**:
   Лучший способ управлять стилями — это вынести их в отдельный файл `.css`. Это позволяет отделить содержание (HTML) от оформления (CSS) и легче управлять стилями на большом проекте.

   Пример подключения внешнего CSS:

   ```html
   <head>
       <link rel="stylesheet" href="styles.css">
   </head>
   ```

Пример содержимого файла `styles.css`:

```css
body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}
h1 {
    color: green;
    font-size: 24px;
    text-align: center;
}
p {
    font-size: 16px;
    margin: 10px 0;
}
```

### CSS-свойства

#### Цвета и шрифты

- `color`: задаёт цвет текста.
- `font-size`: определяет размер шрифта.
- `background-color`: отвечает за цвет фона элемента.

Пример:

```css
h1 {
    color: #ff6347;
    background-color: #f0f0f0;
}
```

#### Отступы и поля

CSS помогает управлять расстоянием между элементами с помощью свойств **`margin`** (внешние отступы) и **`padding`** (внутренние отступы).

Пример:

```css
p {
    margin: 20px; /* Внешний отступ */
    padding: 10px; /* Внутренний отступ */
}
```

### Научпоп оффтоп: Как работает CSS?

CSS называется каскадными таблицами стилей, потому что стили применяются последовательно (или каскадно). Это значит, что если есть несколько правил для одного элемента, браузер применит наиболее специфичное из них. CSS — это не просто язык для оформления; это инструмент, который работает с **иерархией**, **приоритетом** и **наследованием**.

## Заключение

HTML и CSS — это основа любой веб-страницы. HTML задаёт структуру, а CSS — оформление. Их совместное использование позволяет создавать не только функциональные, но и красивые сайты.

### Продолжение лекции: Адаптивная верстка и современные CSS-функции

## Размеры и их роль в адаптивной верстке

Одной из ключевых составляющих успешного веб-дизайна является умение правильно работать с размерами элементов. С развитием технологий важно делать сайты не только красивыми, но и удобными для просмотра на различных устройствах, будь то мобильные телефоны, планшеты или большие мониторы. Поэтому понимание, какие единицы измерения и методы позиционирования применять, имеет огромное значение.

### Единицы измерения в CSS

#### Пиксели (px)

**Пиксели (px)** — это фиксированная единица измерения, которая определяет точное количество пикселей на экране. Пиксели полезны, когда необходимо задать точные размеры элементов и когда макет не зависит от размеров экрана. Однако, их недостаток заключается в том, что такие значения не масштабируются, и на разных устройствах могут возникнуть проблемы с читаемостью или адекватным отображением.

Пример:
```css
.element {
  width: 300px;
  height: 200px;
}
```

#### Проценты (%)

**Проценты (%)** — это относительная единица измерения, которая задаёт размеры элемента в зависимости от размера его родительского контейнера. Она полезна для создания гибких и адаптивных макетов, так как размеры элементов будут изменяться в зависимости от ширины или высоты их родительского блока.

Пример:
```css
.element {
  width: 50%; /* Элемент займет 50% ширины родительского контейнера */
}
```

#### Вьюпортные единицы (vw, vh)

**Вьюпортные единицы** (vw и vh) — это ещё один способ задания размеров элементов, зависящий от размеров окна браузера (вьюпорта). Один **vw** — это 1% от ширины окна, а один **vh** — это 1% от его высоты. Эти единицы особенно полезны для создания макетов, которые должны адаптироваться к экрану пользователя.

Пример:
```css
.element {
  width: 100vw; /* Элемент займет всю ширину окна браузера */
  height: 50vh; /* Элемент займет половину высоты окна браузера */
}
```

### Работа с цветами в CSS

Цвет в веб-дизайне играет важную роль, так как он влияет на восприятие сайта пользователем. CSS предоставляет разнообразные способы задания цветов для элементов, позволяя выбирать самые разные оттенки.

#### Способы задания цвета

- **Ключевые слова** (например, `red`, `blue`, `green`) — простое задание цвета по имени.
  
  Пример:
  ```css
  .element {
    color: red;
  }
  ```

- **RGB** (Red, Green, Blue) — задает цвет в терминах интенсивности красного, зеленого и синего. Это позволяет точно определять нужный оттенок цвета.

  Пример:
  ```css
  .element {
    color: rgb(255, 0, 0); /* Красный */
  }
  ```

- **RGBA** — добавляет альфа-канал, который управляет прозрачностью цвета. Значение альфа-канала варьируется от 0 (полностью прозрачный) до 1 (полностью непрозрачный).

  Пример:
  ```css
  .element {
    color: rgba(255, 0, 0, 0.5); /* Полупрозрачный красный */
  }
  ```

- **HEX-коды** — шестнадцатеричный формат для задания цвета. Это более компактная запись, которая часто используется при разработке.

  Пример:
  ```css
  .element {
    color: #FF5733; /* Оранжевый оттенок */
  }
  ```

- **HSL** — задаёт цвет через оттенок (H), насыщенность (S) и светлоту (L). Это удобная модель для выбора цветов в дизайнерских задачах.

  Пример:
  ```css
  .element {
    color: hsl(360, 100%, 50%); /* Ярко-красный */
  }
  ```

#### Градиенты

Градиенты позволяют создавать плавные переходы между двумя или более цветами. CSS поддерживает линейные и радиальные градиенты.

- **Линейный градиент**:
  ```css
  background: linear-gradient(to right, red, yellow);
  ```

- **Радиальный градиент**:
  ```css
  background: radial-gradient(circle, red, yellow);
  ```

Эти градиенты могут применяться к фону, а также к любому элементу, чтобы добавить глубину и эффект плавного перехода.

### Адаптивные изображения в CSS

Адаптивная верстка требует, чтобы изображения автоматически подстраивались под разные размеры экранов. Для этого в CSS существуют свойства, которые помогают управлять размерами изображений.

- **background-size** — позволяет задавать размеры фонового изображения. Если поставить значение `cover`, изображение будет растягиваться на всю площадь контейнера, сохраняя при этом соотношение сторон.
  
  Пример:
  ```css
  .element {
    background-image: url('image.jpg');
    background-size: cover; /* Фоновое изображение растягивается, чтобы покрыть весь элемент */
  }
  ```

- **background-position** — управляет позиционированием фона внутри элемента. 

  Пример:
  ```css
  .element {
    background-position: center; /* Центрирует фоновое изображение */
  }
  ```

Для адаптивных изображений, которые должны изменять размер в зависимости от ширины экрана, можно использовать свойство **`max-width: 100%`**, чтобы изображение никогда не выходило за пределы родительского элемента.

Пример:
```css
img {
  max-width: 100%; /* Изображение будет масштабироваться до размера родительского контейнера */
  height: auto; /* Высота будет изменяться пропорционально ширине */
}
```

### Прозрачность и альфа-канал

Прозрачность играет важную роль в веб-дизайне, особенно для наложения элементов друг на друга, создания эффектов или стилизации.

- **opacity** — это свойство, которое задаёт общую прозрачность элемента, включая его содержимое.

  Пример:
  ```css
  .element {
    opacity: 0.5; /* Полупрозрачный элемент */
  }
  ```

- **rgba** — для создания полупрозрачного фона или текста можно использовать цвет с альфа-каналом.

  Пример:
  ```css
  .element {
    background-color: rgba(0, 0, 0, 0.3); /* Полупрозрачный черный фон */
  }
  ```

### Блочная модель и отступы

CSS-блочная модель включает в себя такие важные параметры, как **margin**, **padding** и **border**, которые помогают управлять расположением и размерами элементов.

- **Margin** — внешний отступ, который добавляет пространство вокруг элемента, отделяя его от других элементов.
- **Padding** — внутренний отступ, который добавляет пространство внутри элемента между его содержимым и границей.
- **Border** — граница элемента, которая может быть различных толщин и цветов.

Пример:
```css
.element {
  margin: 20px; /* Внешний отступ */
  padding: 15px; /* Внутренний отступ */
  border: 2px solid black; /* Граница */
}
```

Для центрирования блока можно использовать следующее:
```css
.element {
  margin: 0 auto; /* Центрирует элемент по горизонтали */
  width: 50%; /* Ширина элемента */
}
```
